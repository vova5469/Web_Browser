API (Application Programming Interface) - это механизмы, которые позволяют двум программным компонентам взаимодействовать друг с другом,
используя набор определений и протоколов.


Эндпоинт (с англ. конечная точка) представляет собой некий шлюз, который соединяет серверные процессы приложения с внешним интерфесом.
Эндпоинты выполняют конкретную задачу, принимают параметры и возращают данные клиенту.


URL (Uniform Resource Locator) - система унифицированых адресов электронных ресурсов, или единообразный определитель местонахождения ресурса в сети Интернет.
URI (Uniform Resource Identifier) - унифицированый (единообразный) идентификатор ресурса.
URN (Uniform Resource Name) - единообразное название (имя) ресурса.


IP-адрес (Internet Protocol) - цифровой идентификатор, присваеваемый устройству, которое работает в условиях публичной или локальной сети на основе стека протоколов.


Идентификация - установление тождественности неизвесного обьекта извесному на основании совпадения признаков: распознание.
Аутентификация - проверка подлинности предьявленного пользователем идентификатора.
Авторизация - представление определенному лицу или группе лиц прав на выполнение определенных действий, а также процесс проверки(подтверждения) данных прав
при попытке выполнения этих действий.


DNS - компьютерная распределенная система для получения информации о доменах.
Чаще всего используется для получения IP-адреса по имени хоста, получения информации о маршрутизации почты и обслужующих узлах для протоколов в домене.
Основная цель в DNS - это отображение доменных имен в IP-адреса и наоборот IP в DNS.

Откат - это состояние системы -востановление данных компьютерной системы с помощью резервной копии и точки востановления.
